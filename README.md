# Материалы курса

В этом репозитории будут размещаться материалы курса по глубокому обучению, который читался ФИВТ МФТИ в 2019 году.

## Логика работы с репозиторием

* В папке `lecture_notes` располагаются **конспекты лекционных занятий** (обычно в формате `.pdf`), в которые я добавляю помимо непосредственно того, что было на лекциях ещё материал, который кажется мне интересным, в том числе ссылки на дополнительные ресурсы, где можно детальнее изучить тему или её часть.
* В папке `seminars` располагаются **jupyter-ноутбуки семинаров**.

В каждой из папок есть свой readme, в который я добавляю дополнительную инфу по лекциям и семинарам, в том числе ссылки на оригиналы видосов на YT.
